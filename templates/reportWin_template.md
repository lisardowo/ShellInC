# Win Report (Template)

If you found a flag and want to report your win to an Issue, you can do it in this format:

---

## OBTAINED FLAG

```
DVShell{The flag you got}
```

## A BRIEF DESCRIPTION OF HOW YOU DID IT

- Explain in 1 to 3 lines how you exploited the program.
- Example: "I used a buffer of 400 bytes to overflow the buffer with the command X and achieved a RCE."

## PROOF

- A screenshot of the flag in your terminal.
- (Optional) exploit/script/command used.

## (Optional) Aditional Notes

- You found any problems doing this exploit? Some piece of advice to your fellow coders?

---

**Thanks for your efforts!**
